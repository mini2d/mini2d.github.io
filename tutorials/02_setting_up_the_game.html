Setting up the game

Once your webserver is installed and running, you are going to want to create a folder inside the htdocs folder as written in the XAMP/WAMP documentation. Let’s call it “example” for instance.

Writing your index.html file

Next up, navigate inside this folder and create index.html and paste there the following code.

<!doctype html>
<html>
    <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no" />
        <title>Example</title>
    </head>
    <body>
        <script src="mini2d.js"></script>
    </body>
</html>

This is just a basic skeleton for your HTML page. There is not much more to it though.

Create a file for your game source code

The next thing you are going to want to do, is to create a JavaScript file inside your `example` folder.
Let's call the file game.js.

Link the file in your index.html

To link the JavaScript file to your page you have to add this line below the mini2d.js script tag in the index.html file.

<script src="mygame.js"></script>

And thats it! You are ready to start developing your game.
