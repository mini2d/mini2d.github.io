Writing test code

The next thing you are going to want to do is to test your page, whether it is working correctly.

Add the testing code to your game file

Add the following code to your game.js file

window.onload = function(){
    console.log("I work!");
}

Open the page on your local host

Your game should be running on your local web server and should be found at this address http://localhost:8080/example/.

Open up the console (F12) and check if the message was indeed logged in your console.

If so, you have successfully followed the tutorial so far and you are now ready to continue to the next part.
