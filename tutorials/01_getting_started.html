Getting started

In order to start developing your game you’re going to need several things.

Text editor

You are going to need a text editor, because writing code in your notepad sucks!
I recommend using Atom it has everything you are going to need and is free.

The mini2d.js engine

Next up, you are going to need to download the mini2d.js engine. It can be obtained here.

Local web server

You are also going to have a local web server installed. This is because otherwise your web browser would be blocking requests for loading of assets from your filesystem.
You can grab a simple solution for local web server over here windows version and mac version.

You can find more information about how the local web server works on the MAMP/WAMP website. There are other solutions for this, but that goes beyond the scope of this tutorial.
