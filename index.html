Simple HTML5 game engine.

Focused on mobile devices.

Everything this engine does is with mobile devices in mind.

This means that you won’t find hover state on buttons as that is something only desktop devices are capable of. The aim here is to offer a simple API for games that can be ported directly to all major mobile platforms without any trouble along the way using Apache Cordova or other build tools.

If you are looking for API reference then be sure to check out documentation.

If you want to help with getting started and are new to creating games, then be sure to check the official tutorial here.

For those already familiar with the basic concepts of javascript based games could benefit more from interative examples found found here.
