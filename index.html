mini2d.js

Is a mobile first game engine.

Everything this engine does is with mobile devices in mind.

This means that you won’t find a hover state on buttons as that is something only desktop devices are capable of. The aim here is to offer a simple API for games that can be ported directly to all major mobile platforms without any trouble along the way using Apache Cordova or other build tools.

Why mini2d.js

If your goal is to build a simple game and don't want to deal with problems tied to development without a framework.

If you think that other solutions like Phaser.js are too heavyweight solutions for your game.
